APa itu HTML?

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuata halaman web. penanda atau markup ini, nanti akan kita sebut dengan tag. penanda atau markup ini,nanti akan kita sebut dengan HTML berperan untukmenentukan struktur konten dan tampilan dari sebuah web. kalau kita ibaratkan nih, HTML itu seperti batu bata untuk membangun rumah. batu bata ini dapat disusun, hingga menjadi fondasi dasar, HTML memiliki kelebihan lainnya adalah HTML mudah untuk dipelajari oleh seseorang web developer pemula kelebihan lainnya adalah HTML merupakan bahasa pemrograman yang gratis atau open source.
